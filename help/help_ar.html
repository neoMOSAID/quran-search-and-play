1. المقدمة

يمتلك البرنامج المميزات التالية:

  • بحث متقدم في نص القرآن الكريم بخطوط عثمانية ومبسطة
  • تشغيل التلاوات الصوتية مع مزامنة النص
  • نظام متكامل لإدارة الملاحظات والتدبرات
  • واجهة مستخدم قابلة للتخصيص (الوضع الليلي/النهاري)
  • دعم كامل للغة العربية وتصميم RTL
  • إمكانية تصدير واستيراد الملاحظات بصيغة CSV

2. واجهة المستخدم

المكونات الرئيسية:

  1. شريط البحث العلوي:
    • قائمة السور (114 سورة)
    • خيارات طريقة البحث (نصي، سورة، نطاق آيات)
    • حقل إدخال البحث الذكي مع تاريخ البحث
    • أزرار التحكم (مسح، إعدادات)
  2. منطقة النتائج:
    • عرض الآيات المطابقة مع الترقيم القرآني
    • تمرير سلس مع تحميل تلقائي للنتائج
    • إمكانية التحديد والتنقل بالأسهم
  3. منطقة التفاصيل:
    • عرض السياق القرآني (5 آيات قبل وبعد)
    • محرر الملاحظات مع خيارات الحفظ والحذف
    • عرض تاريخ الإنشاء والتعديل للملاحظات

3. أنواع البحث

1.3 البحث النصي:

  • يدعم البحث بالكلمات المفتاحية العربية
  • مثال: البحث عن "الرحمن الرحيم"
  • حساس لحالة التشكيل

2.3 البحث بالسورة:

  • اختيار السورة من القائمة المنسدلة
  • إدخال رقم السورة مباشرة (1-114)
  • مثال: "5" لعرض سورة المائدة كاملة

3.3 البحث بنطاق الآيات:

  • صيغة البحث: [رقم السورة] [الآية الأولى] [الآية الأخيرة]
  • مثال 1: "2 255" لعرض آية الكرسي فقط
  • مثال 2: "2 255 260" لعرض الآيات من 255 إلى 260 من البقرة

4. التلاوة الصوتية

مميزات التشغيل:

  • تشغيل آية محددة (Space)
  • تشغيل نطاق آيات (Ctrl+R)
  • تشغيل سورة كاملة (Ctrl+A)
  • التشغيل التتابعي التلقائي بين السور
  • مؤشر مرئي للآية قيد التشغيل

إدارة الملفات الصوتية:

  1. تحديد مجلد الصوتيات من القائمة (Menu → Set Audio Directory)
  2. صيغة الملفات: SSSAAA.mp3 (SSS = رقم السورة، AAA = رقم الآية)
  3. مثال: 001001.mp3 لسورة الفاتحة آية 1

5. إدارة الملاحظات

العمليات الأساسية:

  • إضافة ملاحظة جديدة (Ctrl+N)
  • حفظ التعديلات (Ctrl+S)
  • حذف الملاحظات (Ctrl+Delete)
  • عرض التاريخ التلقائي للإنشاء/التعديل

المميزات المتقدمة:

  1. عرض الملاحظات حسب تاريخ التعديل
  2. بحث سريع ضمن الملاحظات
  3. معاينة الملاحظات في القائمة الجانبية
  4. تنسيق النص الأساسي في المحرر

7. اختصارات لوحة المفاتيح

الاختصار الوظيفة
Ctrl + F الانتقال إلى حقل البحث
Ctrl + D تبديل الوضع الليلي/النهاري
Ctrl + H فتح دليل المستخدم
Ctrl + J الانتقال إلى سورة الآية المحددة
Ctrl + K العودة إلى سورة التشغيل الحالية
Ctrl + A تشغيل السورة كاملة
Ctrl + P تشغيل الآية مع السياق (5 آيات بعدها)
Ctrl + R تكرار نطاق آيات محدد
Ctrl + S حفظ الملاحظة/إيقاف التشغيل
Ctrl + N ملاحظة جديدة
Ctrl + Delete حذف الملاحظة
Esc تبديل الخط العثماني/المبسط
Backspace العودة إلى قائمة النتائج
